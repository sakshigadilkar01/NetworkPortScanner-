# NetworkPortScanner-
Network Port Scanner GUI 🔍 A GUI-based tool to scan open ports on a target IP or domain. Uses multi-threading for faster and efficient scanning. Helps identify running services and analyze basic network security.
# 🚀 Features
Scan ports for any IP address or domain
Displays open ports with detected services (HTTP, FTP, SSH, etc.)
Start/Stop scan functionality
Save scan results to a file

# 🛠️ Tech Stack
Python
Tkinter (GUI)
Socket Programming
Threading & Queue

# 📌 How It Works
Enter target IP address or hostname
Specify start and end port range
Click Start Scan
View live results and progress
Save results if needed

# ▶️ Run the Project
python portscanner.py
